# Arduino-PY32

Arduino core for PUYA [PY32](https://www.puyasemi.com/cpzx3/info_271_aid_247_kid_246.html) ARM M0+ MCU.


## 文档与上手教程

可以直接查看 [AirMCU 的官方文档](https://arduino.luatos.com/)，包括了接口资料与上手教程。

## Support MCU

| :-------: | :-----------------: |
|  PY32F002A| <https://www.puyasemi.com/> |
